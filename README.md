# githut

a TUI for finding, browsing, and grabbing GitHub repos — without touching a browser.

search GitHub, read READMEs, clone, fork, star. all from the terminal.
built for headless environments, SSH sessions, and people who just prefer the terminal.

built with Rust + ratatui.

> work in progress. nothing works yet.

## usage

```
githut
```

requires `gh` to be installed and authenticated (`gh auth login`).
